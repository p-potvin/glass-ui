# Contributing to Vault Flows

Thank you for helping build the visual layer of the VaultWares AI ecosystem.

## Mandatory Steps

1. Read the [Brand Guidelines](https://raw.githubusercontent.com/p-potvin/vaultwares-docs/main/agents/branding.mdx) and [Agent Knowledge Dissemination System](https://raw.githubusercontent.com/p-potvin/vaultwares-docs/main/agents/knowledge-dissemination.mdx).
2. Follow the exact structure from vaultwares-template.
3. All UI changes must respect Tailwind + vault-themes.
4. Test local bridge and E2E Playwright tests before PR.
5. Maintain local-first privacy in every workflow.

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for community standards.
